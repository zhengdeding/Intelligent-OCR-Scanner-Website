## Build Instructions

* Git clone the project

```
git clone https://github.com/legendary-acp/Intelligent-OCR-Scanner-Website/
```

* Start a virtualenv

```
python3 -m virtualenv .
source bin/activate
```

* Install pip requirements
```
pip install -r requirements.txt
```

* Run the server
```
flask run
```
